PKR Bot - CLI Commands

A Quick Reference for Your Planning Poker Sessions

This page documents the command-line style flags and commands you can use in Microsoft Teams to run planning poker rounds with PKR. If you prefer a more visual approach, use @PKR begin to open an interactive card.

Command Reference

Basic Usage

To invoke a command, type @PKR (or your bot’s name) followed by the command. For example: 

@PKR begin time 60 title "My Round Title"

Make sure Microsoft Teams recognizes your mention of the bot (it should appear in purple text or show the bot’s icon next to @PKR) so that the message is interpreted as a command rather than regular chat text.

Common Commands

Command Description
begin Starts a new planning poker round. Accepts flags like title, time, participants, and more.
history Shows previously completed rounds. Use flags such as limit, startDate, endDate, and legacy.
config Opens an adaptive card with your default poker settings (timer, scale, confidence, etc.).
help Displays a help card with quick action buttons.

Available Flags

Many flags can be combined. For example, @PKR begin title "Some Ticket" time 60 sets a custom title and 60-second timer.

Flag Purpose Example Usage
title Sets a round title (e.g. referencing a user story or ticket). @PKR begin title "User Access Reviews"
time Sets the timer (seconds) for how long the round stays open. Default is 30s. @PKR begin time 60
participants (or players) Override the participant count if Teams membership can’t be fetched automatically. @PKR begin participants 5
limit / history Shows how many past rounds to retrieve (when using @PKR history). @PKR history 20 or @PKR history limit 10
startDate / endDate Filter rounds by date range (format: YYYY-MM-DD). @PKR history startDate 2025-06-01 endDate 2025-06-30
limitToThisChat For history, only return rounds from the current conversation ID. @PKR history limitToThisChat true
legacy Search the older (v1) data in pkr-legacy. Typically used with history. @PKR history legacy

Combining Flags

You can combine multiple flags, for example: 

@PKR begin title "Backlog Grooming" time 45 participants 4

The bot will parse all flags and start a new round accordingly.

Further Help

Type @PKR help at any time for a quick help card. If you need additional guidance, see our Support page or Homepage for more details.
