PKR Bot: CLI Flags Reference

PKR Bot supports a few command-line style flags when you type @PKR history in your Microsoft Teams chat. Only the following flags are currently functional:

Flag Description Example Usage
limit Show up to this many rounds (default is 10). @PKR history limit 20
legacy Includes older v1 (pre-2025) data from the pkr-legacy table in the results. This also forces the search to the current conversation ID. @PKR history legacy

Examples

  • Show your last 15 rounds:
    @PKR history limit 15
  • Retrieve legacy (v1) data only:
    @PKR history legacy
  • Combine both flags:
    @PKR history legacy limit 5

Note: Previously documented flags such as startDate, endDate, limitToThisChat, and searchTitle are not currently supported. They’ve been removed from this reference.
