How to Use PKR Bot — Detailed Instructions

Follow these steps to run your first planning poker round in Microsoft Teams using PKRbot. Below, you’ll also see how to configure the bot’s “reply location” (i.e., whether the configuration cards appear in your personal chat or the current group chat).

1) Add PKR Bot to Your Chat or Team

If PKR is not already installed, simply click the + icon (or “Get Bots”) in your Team or chat, search for “PKR”, and add it.

[Placeholder Image]
Screenshot showing how to add PKR Bot

2) Start a Round

In a Teams channel or group chat where PKRbot is present, type:

@PKR begin

PKRbot will post a “Round Configuration” card letting you choose options like the Estimation Scale (points, T-shirt, Fibonacci), the round timer, and so forth. Then click Start Round to begin.

[Placeholder Image]
Example of the round config card in Teams

3) Everyone Submits Estimates

PKRbot posts a “Submissions” card listing all participants. As each teammate submits an estimate (via the Estimation Card PKRbot also provides), you’ll see a checkmark (or a filled circle) next to their name.

[Placeholder Image]
Sample submissions tracker card with user statuses

Once all participants have submitted—or the timer expires—PKRbot will automatically reveal final results in a separate “Results” card.

4) View Your Results

PKRbot calculates the mean and standard deviation for numeric scales. For T-shirt sizing, it simply displays what everyone selected. You can optionally post the results back to Jira (if connected).

[Placeholder Image]
Results card with mean, std dev, etc.

Choosing Where the Bot Sends Configuration Cards

PKRbot has a “reply location” setting, determining where it sends configuration-related cards (like the user settings or history search). You can set this in your @PKR config card, under “Bot Reply Location”.

Option What Happens?
Current Chat PKRbot posts config cards in the same channel or group chat you’re currently in.
  • Other users can see these cards, but only you can submit them.
  • Useful if your entire team wants to see changes you make (like enabling confidence fields, etc.).
Personal Chat PKRbot will spin up (or reuse) a 1:1 private chat with you to show config or search cards.
  • No one else sees these personal cards or changes you make to your settings.
  • The actual estimation round is still posted in the group chat, only the config changes remain private.
[Placeholder Image]
Example of user config card that can appear in your personal chat

Additional Tips

  • Type @PKR help at any time to see a quick menu of commands (e.g., Begin, Config, History).
  • If you need to re-estimate a round you already finished, find that round via @PKR history and click the “Re-estimate Ticket” button in the results card.
  • Connect PKRbot to your Jira instance (via @PKR config) to easily post final estimation results to a Jira ticket.
