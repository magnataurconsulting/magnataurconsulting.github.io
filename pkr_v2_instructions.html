PKR Bot - Usage Instructions

Getting Started

1) Add PKR Bot to Microsoft Teams
Use the Teams app store or “Add apps” in your team chat to add PKR to your desired group chat or channel. Then mention @PKR in a chat message to confirm it’s recognized.

2) Begin a Round
Type @PKR begin in a group chat. You should see a card with an estimation scale, time limit, and optional confidence input. Everyone in the chat can then submit an estimate.

3) Review Results
When the timer ends (or all participants have submitted), PKR Bot reveals aggregated results: the mean, standard deviation, and optionally confidence stats. Use these stats to decide your final story point (or time) estimate.

Advanced Configuration

PKR Bot supports custom defaults and flags for more complex sessions. For example, you can set the default estimation scale to Fibonacci or T-shirt sizes, or enable anonymous voting. Type @PKR config to open a settings card. You can also link PKR Bot to Jira for direct posting of results to your issue tickets.

Optional CLI Flags

If you prefer command-like usage, you can add flags to begin or history commands. Example: @PKR begin title "Backend Overhaul" time 60 sets a custom title and a 60-second timer. See the full CLI Command Reference for details on all supported flags.

Need More Help?

At any time, type @PKR help to see a quick help card with shortcuts. For additional troubleshooting, visit our Support page.
